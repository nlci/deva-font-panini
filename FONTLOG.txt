FONTLOG
Panini, Kautilya, Maurya - NLCI Devanagari fonts
================================================

This file provides detailed information on the NLCI Devanagari fonts.
This information should be distributed along with the NLCI Devanagari fonts and any derivative works.

Basic Font Information
----------------------

Pan Devanagari font designed to support all the languages using the Devanagari script.
