NLCI Devanagari fonts
=====================

This file describes the source files included with the NLCI Devanagari fonts.
This information should be distributed along with the NLCI Devanagari fonts and any derivative works.

nepali_chars.lst      - Nepali specific build file
master.gdl            - Graphite rules
features.gdh          - Graphite rules for features
master.sfd            - OpenType rules
other .sfd files      - glyph outline data for each font
panini_unicode.xml    - mapping from legacy to Unicode as used by ttfbuilder
DEV Panini 2 Unicode CONV chart.xls - source for panini_unicode.xml
